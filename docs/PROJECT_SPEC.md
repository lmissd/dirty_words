# 文明用语机器人项目总规范

## 项目名称

文明用语机器人（Civil Language Robot）

## 项目目标

开发一个基于 Raspberry Pi 4B 的桌面 AI 机器人。机器人能够通过语音与用户交互，对用户语言进行文明程度检测，并对不文明表达进行提醒和改写建议。

项目采用云端大模型 API 方案，不在树莓派本地运行大模型。

## 开发原则

- 模块化设计
- 易维护
- 易扩展
- 面向未来升级
- 所有配置独立存放
- 所有 API 密钥统一管理
- 所有功能组件解耦

## 硬件环境

- 主控：Raspberry Pi 4B
- 系统：Raspberry Pi OS 64-bit
- 主机名：yuangungun
- 用户：pi
- 网络：WiFi 联网
- SSH：开启

## 硬件组成

- 输入设备：双麦克风阵列，USB 麦克风兼容方案
- 输出设备：喇叭，功放模块
- 显示设备：HDMI 显示屏
- 存储：MicroSD 卡

## 核心功能

- 语音唤醒：长期待机，持续监听唤醒词，默认“小文小文”，未来支持多个唤醒词。
- 录音：唤醒成功后录制 5-10 秒临时音频。
- 语音识别：调用云端语音识别服务，推荐 OpenAI Speech-to-Text。
- 文明用语分析：调用大模型 API，返回 JSON。
- 屏幕显示：大字体、中文优先、全屏显示，适合触摸屏扩展。
- 语音播报：调用 TTS 接口播报分析结果和改写建议。

## 文明分析 JSON 返回格式

```json
{
  "civilized": false,
  "score": 20,
  "reason": "存在侮辱性表达",
  "suggestion": "请尝试使用更加尊重对方的表达方式"
}
```

## 系统状态机

```text
待机
↓
监听唤醒词
↓
被唤醒
↓
录音
↓
语音转文字
↓
文明分析
↓
显示结果
↓
语音播报
↓
返回待机
```

## 第一阶段目标

实现：

1. 唤醒词检测
2. 录音
3. 语音识别
4. GPT 文明分析
5. 屏幕显示
6. 语音播报

确保能够形成完整闭环。
